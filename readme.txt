検索拡張ツール使用方法

〇インストール
1. fileCp.ps1を実行
　実行することにより、検索拡張ツールがエクセルアドインの格納場所にコピーされます。
　格納場所：C:\Users\<ユーザ名>\AppData\Roaming\Microsoft\AddIns
　コピーに失敗した場合は「copy_ng」が作成されます。
　処理終了後、格納場所のショートカットリンクが作成されます。
　
2. エクセルを起動
3. エクセル画面の「ファイル」タブ→オプション→アドインの順に移動
4. 画面下部、「管理」が「EXCEL アドイン」の状態で右の「設定」をクリック
5. 「Searchextensionaddin」にチェックを入れ「OK」をクリック
6. 任意のタブにカーソルを当てて右クリック→リボンのユーザ設定
　　コマンドの選択でマクロを選択→以下4つを追加
　◆ブック内検索
　　・searchForm
　　・replaceForm
　◆ファイル検索
　　・searchFileForm
　　・replaceFileForm
完了

〇使用方法
・「インストール」で追加したボタンをクリックするとポップアップが出ます。
　◆ブック内図形検索・置換
　　・使用感は通常の検索画面と同様でテキストボックスに文字をいれ、実行したいボタンをクリックしてください。
　　・「searchForm」は検索、「replaceForm」は置換となります。
　　・ショートカットキーとして「searchForm」はCtrl + shift + s、「replaceForm」はCtrl + shift + Rで使用できます。
　◆ファイル図形検索・置換
　　・使用感は通常の検索画面とサクラエディタのGrep機能を足した感じになってます。
　　・「searchFileForm」はファイル検索、「replaceFileForm」はファイル置換です。
　　・検索場所は「・・・」ボタンからフォルダを選択可能です。(直接入力も可能、パスの最後に「\」は不要)
　　・対象ファイルは「;」区切りで複数設定可能です。
　　・検索結果は「【図形検索】検索結果」シート、置換結果は「【図形検索】置換結果」シートが追加されます。
　　　(既に存在する場合はシートの内容を削除する)

○注意事項
　・置換実行時はエクセルの仕様上、置換前に戻す(Ctlr + z)ができなくなります。
　　「すべて置換」する場合は事前にバックアップをとることをお勧めします。
　・ファイル検索の「対象ファイル」は正規表現として検索されます。
　　特殊文字を使用する際は注意してください。(正規表現で使われる記号はエスケープ文字「\」が必要)
　　※記号が無ければ支障は出ません。
　　　例：
　　　　○：test.xlsx;test.*\.xlsx
　　　　×：*.xlsx ←ワイルドカードとしては使用できない。
