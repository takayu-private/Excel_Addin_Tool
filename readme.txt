検索拡張ツール使用方法

〇インストール
1. fileCp.ps1を実行
　実行することにより、検索拡張ツールがエクセルアドインの格納場所にコピーされます。
　格納場所：C:\Users\<ユーザ名>\AppData\Roaming\Microsoft\AddIns
　コピーに失敗した場合は「copy_ng」が作成されます。
　処理終了後、格納場所のショートカットリンクが作成されます。
　
2. エクセルを起動
3. エクセル画面の「ファイル」タブ→オプション→アドインの順に移動
4. 画面下部、「管理」が「EXCEL アドイン」の状態で右の「設定」をクリック
5. 「Searchextensionaddin」にチェックを入れ「OK」をクリック
6. リボンのユーザ設定(任意のタブ上で右クリック)で以下2つを追加
　　・searchForm
　　・replaceForm
完了

〇使用方法
・「インストール」で追加したボタンをクリックするとポップアップが出ます。
　使用感は通常の検索画面と同様でテキストボックスに文字をいれ、実行したいボタンをクリックしてください。
　「searchForm」は検索、「replaceForm」は置換となります。
・ショートカットキーとして「searchForm」はCtrl + shift + s、「replaceForm」はCtrl + shift + Rで使用できます。
